# スイカゲーム(Unity) ミニプロジェクト

このフォルダには、2D物理で動くスイカゲームの最小実装が入っています。  
シーン内のGameObjectは自動生成されるので、手動で配置は不要です。

## 使い方
1. Unityでこのフォルダを開く
2. 新規シーンを作成して保存 (任意)
3. 再生 (左クリック/Spaceで落下)

## 操作
- マウス移動: 落下位置の移動
- 左クリック or Space: 果物を落とす
- R: ゲームオーバー後にリスタート

## カスタム
- `GameManager`の`fruitDefinitions`でサイズ・色・スコアを変更可能
- `spawnTypeCount`で初期に出てくる果物の種類数を変更
